## ✨ 新增功能

**核心能力**

- **插件**：从工作区 `plugins/` 目录安装扩展 ([#3101](https://github.com/agentscope-ai/CoPaw/pull/3101), [#3131](https://github.com/agentscope-ai/CoPaw/pull/3131), [#3132](https://github.com/agentscope-ai/CoPaw/pull/3132))。
- **`copaw task`**：无需启动 Web 服务，在终端执行一次性任务 ([#3031](https://github.com/agentscope-ai/CoPaw/pull/3031))。
- **对话中的 `/model`**：不必打开设置页，即可切换模型、查看可用模型、恢复默认或查看模型信息 ([#3133](https://github.com/agentscope-ai/CoPaw/pull/3133))。

**模型与提供商**

- **SiliconFlow**：内置 SiliconFlow OpenAI 兼容 API 的中国区与国际区端点，开箱即用 ([#2886](https://github.com/agentscope-ai/CoPaw/pull/2886))。
- **CoPaw Local**：支持图像与视频能力；控制台可调项更丰富；在 Windows 上模型下载与能力探测更可靠 ([#3021](https://github.com/agentscope-ai/CoPaw/pull/3021), [#3087](https://github.com/agentscope-ai/CoPaw/pull/3087), [#3140](https://github.com/agentscope-ai/CoPaw/pull/3140))。

**安全**

- **敏感凭据存储**：API Key 等敏感值落盘加密；在系统支持时，加密主密钥优先存入操作系统钥匙串 ([#3025](https://github.com/agentscope-ai/CoPaw/pull/3025))。

**控制台与界面**

- **聊天输入历史**：用方向键翻阅此前的用户输入行 ([#2466](https://github.com/agentscope-ai/CoPaw/pull/2466))。
- **聊天搜索**：跨会话检索消息正文 ([#2842](https://github.com/agentscope-ai/CoPaw/pull/2842))。
- **置顶会话**：将会话固定在列表顶部 ([#3137](https://github.com/agentscope-ai/CoPaw/pull/3137))。
- **按智能体记忆会话**：切换智能体时，自动回到该智能体上次打开的聊天 ([#3155](https://github.com/agentscope-ai/CoPaw/pull/3155))。
- **界面图标**：为内置工具提供图标，并在模型选择器旁展示提供商标识 ([#3061](https://github.com/agentscope-ai/CoPaw/pull/3061), [#3130](https://github.com/agentscope-ai/CoPaw/pull/3130))。

**技能与工具**

- **技能指令**：`/skills` 列出当前频道已启用的技能；输入 `/<技能名>` 可查看说明或携带消息调用该技能 ([#3150](https://github.com/agentscope-ai/CoPaw/pull/3150))。
- **技能池标签**：使用标签整理共享技能池 ([#2837](https://github.com/agentscope-ai/CoPaw/pull/2837), [#3069](https://github.com/agentscope-ai/CoPaw/pull/3069))。
- **MCP 工具发现**：通过 HTTP API 查询已连接 MCP 服务端暴露的工具名称、说明与参数 ([#3149](https://github.com/agentscope-ai/CoPaw/pull/3149))。

**频道**

- **QQ**：各类消息类型下，文件与富媒体发送表现一致 ([#3012](https://github.com/agentscope-ai/CoPaw/pull/3012))。
- **企业微信**：入站消息保留引用与回复上下文，便于模型理解所指内容 ([#3024](https://github.com/agentscope-ai/CoPaw/pull/3024))。

## 🔄 优化改进

- **时区选择**：时区选项名称随当前界面语言展示 ([#2497](https://github.com/agentscope-ai/CoPaw/pull/2497))。
- **文件大小**：控制台内统一使用易读的字节单位 ([#2808](https://github.com/agentscope-ai/CoPaw/pull/2808))。
- **控制台启动**：较重的设置类页面改为按需加载，首屏打开更快 ([#3122](https://github.com/agentscope-ai/CoPaw/pull/3122))。
- **大型技能列表**：技能池条目很多时，滚动与操作更顺滑 ([#3141](https://github.com/agentscope-ai/CoPaw/pull/3141), [#3158](https://github.com/agentscope-ai/CoPaw/pull/3158))。
- **提供商连通性测试**：测试结果提示已随界面语言本地化 ([#2913](https://github.com/agentscope-ai/CoPaw/pull/2913))。
- **错误码**：各聊天频道、HTTP API 与 CLI 之间的错误码更加一致 ([#3110](https://github.com/agentscope-ai/CoPaw/pull/3110))。
